**Ever wondered what really happens when you render something through a portal in React Native?**

In my talk **The Illusion of Portals in React Native** at [App.js Conf](https://appjs.co/), we look beyond the familiar **Portal** API to understand how different implementations actually work, how they affect the React tree, and what a true portal should look like internally — from the illusion created at the React layer to native portals, view re-parenting, and the new possibilities they unlock.
